# Módulo AR · HUB COMLURB

Pasta pronta para subir no GitHub Pages.

## Estrutura

```txt
ar/
├── index.html
├── ar.css
├── ar-config.js
└── ar.js
```

## Como publicar

1. Suba a pasta `ar` na raiz do repositório `COMLURB`.
2. Confirme que o repositório já possui:
   - `assets/css/hub-premium.css`
   - `assets/logos/RIOPREFEITURA Comlurb horizontal monocromatica branco.png`
3. Acesse:

```txt
https://urbanflowrio.github.io/COMLURB/ar/
```

## Atualização dos dados

Os links CSV ficam no arquivo `ar-config.js`.

Quando a planilha do Google Sheets for atualizada mensalmente, o painel passa a ler os dados publicados automaticamente.
